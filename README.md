#### example of usage

```bash
Input dimension you want work with:
3
Input polynomial you want to shrink:
x1+x2+x3+!x1x2+!x1!x3+!x2!x3+x1!x2x3+!x1!x2!x3
Your polynomial has been shrinken to this form:
x3+!x1!x3
```
